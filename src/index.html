    The goal of this assignement is to code a simple game that the user interacts and draws some geometrical shapes with.

    Features:

    • Mobile First.
    • ES6 Vanilla Javascript.
    • Native JavaScript modules on the browser.
    • CSS variables.
    • Harnessing the power of the latest JavaScript and CSS feature without the need for pre-processors on modern browsers.
    • The only development dependency is http-server which is required when using <script type="module">.

    How to play:

    Click on 3 spots inside the marked space below and see the shapes and their coordinates. You can drag the shapes along the space and the coordinates will be updated. The game has a button on the side that you can clear the points with. 

    Enjoy!
